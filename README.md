# life-timeline
## Getting Started

1. Create network

```console
$ docker network create lifetimeline_default
```

2. Build docker image

```console
$ docker compose build
```

3. Run the development server

```consle
$ docker compose up
```

4. Open [http://localhost:3000](http://localhost:3000)

## For Developpers
- Node.js: 18.16.0
- npm: 9.8.1
